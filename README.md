## 🔵Goal
To build strong networking fundamentals for cybersecurity by understanding how data travels across networks.

## 🔄 Transmission Methods

- **Simplex**
  - One-way communication
  - Example: Keyboard → Computer

- **Half Duplex**
  - Two-way but one direction at a time
  - Example: Walkie-talkie

- **Full Duplex**
  - Two-way simultaneous communication
  - Example: Phone call

---

## 📊 Network Criteria

A good network must satisfy:

- **Reliability**
  - Ensures data is delivered correctly and consistently

- **Performance**
  - Measures speed (throughput, latency, delay)

- **Security (CIA Triad)**
  - **Confidentiality** → Only authorized users can access data  
  - **Integrity** → Data is not altered  
  - **Availability** → Data is accessible when needed

- Star  
- Ring  
- Tree

---

## 🔷 Point-to-Point

- Direct connection between two devices
- Simple and fast

---

## 🕸️ Mesh Topology

- Each device connects to every other device

### 📌 Formula:
- Links = n(n - 1) / 2  

### ✅ Advantages:
- High reliability  
- Easy fault identification  
- Less traffic (dedicated links)

### ❌ Disadvantages:
- Expensive  
- Difficult installation  

---

## ⭐ Star Topology

- All devices connected to a central hub/switch
